## Packages
react-konva | Canvas-based blueprint editor
konva | Core canvas library required by react-konva
recharts | Data visualization for cost breakdown
framer-motion | Smooth transitions between wizard steps
clsx | Class name utility
tailwind-merge | Class name merging utility
jspdf | Exporting reports (optional but good for construction apps)

## Notes
The Blueprint Editor uses HTML5 Canvas via Konva.
API endpoints assume strict JSON structures defined in schema.
